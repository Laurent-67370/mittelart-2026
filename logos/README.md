# Logos des Partenaires

Ce dossier contient les logos officiels des partenaires de Mittelart 2026.

## Images requises :

Veuillez placer les fichiers suivants dans ce dossier :

1. **credit-mutuel.png** - Logo Crédit Mutuel (fond blanc, ~400x200px)
2. **le-geant.png** - Logo Le Géant des Beaux-Arts (fond blanc, ~400x200px)
3. **steinmetz.png** - Logo Groupe Steinmetz (fond blanc, ~400x200px)  
4. **cg-photogravure.png** - Logo CG Photogravure (fond blanc, ~400x200px)
5. **mittelhausbergen.png** - Logo Ville de Mittelhausbergen (fond blanc, ~400x200px)

## Format recommandé :
- Format : PNG avec fond transparent ou blanc
- Résolution : 400-600px de largeur
- Poids : < 100 Ko par image (optimisé)

Le logo de la Collectivité Européenne d'Alsace reste en SVG (déjà intégré).
