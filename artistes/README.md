# Dossier des photos d'œuvres d'artistes

## Instructions

Placez les photos des œuvres d'artistes dans ce dossier.

### Nommage des fichiers
Utilisez le format : `nom-prenom.jpg` (en minuscules, avec tirets)

Exemples :
- `jung-daniel.jpg`
- `helwig-jean-jacques.jpg`
- `guerrier-eve.jpg`
- `beiu-stefan.jpg`
- `grusevan-victor.jpg`
- `di-scala-brigitte.jpg`
- `roth-rene.jpg`
- `desjardins-jordane.jpg`
- `perret-roland.jpg`
- `gouvion-sophie.jpg`

### Format recommandé
- **Format** : JPG ou PNG
- **Taille** : Minimum 400x400 pixels
- **Poids** : Maximum 200 Ko par image
- **Optimisé pour le web** : Utilisez https://tinypng.com/

### Après ajout des images
Consultez le fichier `GUIDE_PHOTOS.md` à la racine du projet pour savoir comment intégrer les images dans les cartes d'artistes.
